# VA CodeEditor for ACF

これは、WordPressのプラグイン Advanced Custom fieldsのadd-onです。

このプラグインをWordPressにインストールし有効化すると、

4種類のソースコードエディターを追加できるようになります。

使用ライブラリとして、CodeMirror、Emmetを使用しています。
